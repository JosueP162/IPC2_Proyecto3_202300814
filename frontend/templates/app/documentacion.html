Sistema de Gestión Cloud - Tecnologías Chapinas, S.A.

Descripción del Proyecto

Sistema integral para la gestión y facturación de servicios de infraestructura en la nube.

Arquitectura

  • Backend: Flask API REST con base de datos XML
  • Frontend: Django con AdminLTE
  • Comunicación: APIs HTTP/JSON

Funcionalidades Principales

  • Gestión de recursos, categorías y configuraciones
  • Registro de clientes e instancias
  • Carga de datos mediante XML
  • Proceso de facturación automática
  • Generación de reportes PDF
  • Análisis de ventas

Tecnologías Utilizadas

  • Python 3.8+
  • Flask + Flask-CORS
  • Django 4.2+
  • AdminLTE 4.0
  • ReportLab (PDF)
  • XML.etree
